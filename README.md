# 💻 n8n Custom Code Solutions

Stop struggling with complex data transformations! This repository provides copy-paste friendly code snippets for the **n8n Code Node** (JavaScript/Python).

## 🚀 Why use these snippets?
n8n is powerful, but sometimes standard nodes aren't enough. These scripts help you:
* Clean messy data.
* Format dates and currency.
* Filter complex JSON arrays.

---

## 🛠️ Featured Code Snippets

### 1. Advanced Date Formatter (JavaScript)
Convert any timestamp into a human-readable format (e.g., "7th April 2026").
* **Use Case:** Sending professional emails or reports.
* **File:** `/javascript/date-formatter.js`

### 2. Remove Duplicates from Input (Python)
A simple script to ensure your workflow doesn't process the same item twice.
* **Use Case:** Cleaning lead lists or database entries.
* **File:** `/python/remove-duplicates.py`

### 3. Image URL to Base64
Convert an image link directly into Base64 format for API uploads.
* **Use Case:** Sending images to OpenAI Vision or custom APIs.
* **File:** `/javascript/image-to-base64.js`

---

## 📖 How to Use
1.  Navigate to the folder (JavaScript or Python).
2.  Copy the code from the `.js` or `.py` file.
3.  In your n8n workflow, add a **Code Node**.
4.  Paste the code and connect your data.
